Registro de Usuario con Validación en JavaScript

Proyecto de registro de usuarios con validación completa en JavaScript.

Características
- Validación de nombre, email y contraseña
- Contraseña mínima de 8 caracteres
- Validación de formato de correo electrónico
- Mensajes de éxito o error
- Diseño responsivo

Tecnologías usadas
- HTML5
- CSS3
- JavaScript
